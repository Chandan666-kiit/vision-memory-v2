Screenshot placeholders for the Vision & Memory presentation.

Generated by create_presentation.py:
  - screenshot1_upload.png   → Upload screen
  - screenshot2_analysis_form.png → Analysis + name form
  - screenshot3_greeting.png → Greeting output

To use your own screenshots:
  1. Run the Streamlit app and take 3 screenshots.
  2. Save them with the same names above in this folder (ppt_assets).
  3. Run: python create_presentation.py
  4. The new PPT will include your images.
